Instructions to run the code:
1. For example you want to run example_6_Aggregation on Xcelium.
2. Now you need to set your current directory to example_6_Aggregation.
3. Now enter xrun [filename.sv] -access +rwc in case of multiple files list all of them with extension.
4. If you want to see the simulation in gui mode you need to enter this command.
   % xrun [filename.sv] -access +rwc -gui.
